# 回望灯火阑珊 · 写作风格指南

## 概述

本skill定义了《回望灯火阑珊》的写作风格体系，核心特征为**"温而不腻，淡而有味"**——用生活流笔法书写青春与奋斗，以克制的情感表达营造余韵悠长的阅读体验。

## 风格核心：三个关键词

| 关键词 | 含义 |
|-------|------|
| **温** | 叙述视角温和，像老朋友在旁观，不冷漠也不过度介入 |
| **淡** | 情感表达克制，该煽情时偏不煽情，留白让读者自己体会 |
| **真** | 细节真实接地气，不追求戏剧冲突，靠生活质感打动人 |

---

## 叙事规范

### 视角选择

采用**第三人称有限视角**，主要跟随主角视线展开：

- 可以进入主角内心，但保持适度距离
- 不使用全知视角评判人物
- 偶尔可切换到其他人物视角，但需自然过渡

### 节奏把控

**慢热型叙事**——像绿皮火车，不急不躁，晃晃悠悠往前走：

- 舍得花笔墨写"无用"的细节（环境、声音、气味）
- 情感线铺垫可以漫长，不急于推进
- 高潮点轻拿轻放，大事化小，举重若轻

### 场景转换

使用 `---` 分隔符划分场景，每个场景聚焦一个核心事件或情绪。

## 语言规范

### 对话风格：鲜活贫嘴

日常对话要有**京味幽默**和**年轻人的贫嘴**：

```markdown
# 好的例子
"你这形象，拎杯水就是处长，换个皮包就是相亲中介。"
"凯迪拉克都不上，这觉悟，堪比少林俗家弟子。"
"我看你们一个个不考研，都该去写八卦专栏，肯定火。"

# 避免
过于书面化的对话
没有个性特征的对话
所有人说话方式雷同
```

### 内心独白：诗意克制

内心描写时语言要**沉下来**，简洁干净，带淡淡哲思但不说教：

```markdown
# 好的例子
"每个人都在去往某个目的地，只是有些人知道为什么，有些人不知道。"
"也许人生就像这列车——有时候慢、有时候颠簸，但不管多难熬，总得往前走。"

# 避免
大段的心理分析
过于直白的情感宣泄
说教式的人生感悟
```

### 描写风格：白描为主

人物和场景描写采用**白描手法**，用具体细节代替抽象形容：

```markdown
# 好的例子
"她穿着一件淡蓝色的长袖短款外套，一条黑色的修身牛仔裤，裤腿有一点点高出了鞋面，露出白袜子黑皮鞋。"

# 避免
"她美丽动人，气质出众，让人一见倾心。"
```

## 情感表达规范

### 核心原则：欲说还休

**越是重要的情感，越要轻轻放下。**

### 具体技巧

| 场景 | 处理方式 |
|-----|---------|
| 悲伤 | 不写哭泣，写"心像被绞一样的痛"，然后戛然而止 |
| 告别 | 不写豪言壮语，写一个简单的动作或眼神 |
| 暧昧 | 不写表白，写一个小物件、一句双关语 |
| 感动 | 不写热泪盈眶，写"笑了，眼中泪光闪闪，但也在笑" |

### 例子

```markdown
# 告别场景 - 好的写法
所有人把叠在一起的手往上一甩，都感到如释重负。
浩翔笑了，吴彬也笑了，邓娜眼中泪光闪闪，但也在笑。
李礼也笑了。

# 避免的写法
"我们一定会成功的！""大家加油！""我会想你们的！"
（过于直白的情感宣泄）
```

## 人物塑造规范

### 群像素描法

每个人物用**一两个视觉符号**快速锁定形象：

| 人物类型 | 标志性元素示例 |
|---------|---------------|
| 精致型 | 白衬衫永远笔挺、眼镜擦得反光 |
| 热闹型 | 荧光笔、丸子头、大嗓门 |
| 沉稳型 | 冲锋衣、某件随身物品、特殊口音 |
| 温柔型 | 服饰细节、走路的声音、说话方式 |

### 人物语言个性化

每个人物的说话方式要有辨识度：

```markdown
- 吴彬：官腔、一本正经地说俏皮话
- 邓娜：大嗓门、爱用夸张比喻
- 浩翔：广普、温温的语气、不急不慢
- 王晓斌：贼兮兮、爱起哄、光棍气质
```

## 时代质感植入

### 细节类型清单

在叙事中自然融入时代符号，不刻意卖弄：

| 类别 | 可用元素 |
|-----|---------|
| 通讯 | 诺基亚手机、发短信、打电话、拨号上网 |
| 交通 | 绿皮火车、破自行车、公交车 |
| 娱乐 | 当年流行的歌曲、电视剧、KTV |
| 场景 | 大学自习室、食堂、水木论坛、出租屋 |
| 物价 | 符合年代的工资水平、消费水平 |

## 章节结构规范

### 标题风格

章节标题采用**短句命名**，简洁有力：

```markdown
# 好的标题
重返北京、唐晚、圣诞节、走出考场、长城上的牵手

# 避免
过长的标题
带有剧透性质的标题
过于文艺晦涩的标题
```

### 章节内部结构

```markdown
---
title: "章节号.章节名"
date: YYYY-MM-DD
weight: 章节号
draft: false
---

# 章节号.章节名

[开场：场景/氛围建立，100-200字]

---

[主体内容，用 --- 分隔不同场景]

---

[收尾：情感落点或悬念，简洁有力]
```

## 写作检查清单

完成一章后，对照以下清单检查：

- [ ] 对话是否鲜活有个性，避免书面化？
- [ ] 情感高潮处是否克制，有没有过度煽情？
- [ ] 是否有具体的时代细节和生活质感？
- [ ] 人物是否有标志性特征，能否一眼辨认？
- [ ] 内心独白是否简洁，有没有变成说教？
- [ ] 场景转换是否自然，节奏是否舒适？
- [ ] 收尾是否留有余韵，而非把话说尽？
