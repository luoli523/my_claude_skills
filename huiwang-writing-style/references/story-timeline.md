# 《回望灯火阑珊》故事时间线

## 时间背景

- **故事主线时间**：2005年9月 - 2006年1月（考研备战期）
- **回忆时间线**：初中 → 大学 → 2004年分手

---

## 已发生事件（第1-28章）

### 前史（通过回忆呈现）

| 时间 | 事件 |
|-----|------|
| 初中时期 | 李礼和唐晚同班，成为前后桌，互生好感 |
| 高中/大学 | 两人异地恋（李礼在北京，唐晚在天津） |
| 2004年末 | 唐晚提出分手 |
| 2004年11月某晚 | 李礼去天津挽回，在宿舍楼下看到唐晚和新男友 |
| 2005年1月 | 第一次考研，李礼失败 |
| 2005年春节后 | 李礼在老家蹲了大半年 |
