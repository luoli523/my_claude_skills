# 《回望灯火阑珊》人物档案

## 主要人物

### 李礼（主角）

| 属性 | 描述 |
|-----|------|
| 年龄 | 23岁（2005年） |
| 籍贯 | 湖南长沙 |
| 学历 | 北京林业大学本科毕业 |
| 目标 | 考研（北航计算机） |
| 标志 | 破旧的《火星考研英语词汇大纲》、会修电脑 |

**性格特点**：
- 表面随和幽默，内心敏感细腻
- 经历过失恋创伤，但在努力走出来
- 重情义，对朋友真诚
- 有小聪明，但不是天才型

**语言风格**：
- 爱开玩笑但不过分
- 内心独白时会有诗意的思考
- 偶尔会说粤语（因为唐晚是广东人）

**情感线**：
- 唐晚（初恋，已分手，仍未完全释怀）
- 徐皙（新的感情萌芽）

---

### 唐晚（李礼初恋）

| 属性 | 描述 |
|-----|------|
| 年龄 | 与李礼同龄 |
| 籍贯 | 广东 |
| 学历 | 天津某大学在读 |
| 现状 | 已有新男友 |

**性格特点**：
- 恬静从容，像《致青春》里的阮莞
- 爱的时候死心塌地，翻篇时也翻得彻底
- 文秀安静，是人群中最耀眼却最低调的那个

**在故事中的存在方式**：
- 主要通过李礼的回忆出现
- 是"不在场"的重要人物
- 象征李礼需要告别的过去

---

### 徐皙（李礼新感情线）

| 属性 | 描述 |
|-----|------|
| 年龄 | 约22岁 |
| 籍贯 | 南方（具体未明） |
| 学历 | 北京林业大学园林系，大四 |
| 身份 | 设计院实习生 |

**外貌特征**：
- 长发披肩，大眼睛细眉毛
- 气质像南方姑娘，干净利落
- 标志：圣诞老人发卡

**性格特点**：
- 落落大方，不扭捏
- 主动但不强势
- 说话带笑，眼睛会眯成小弯

**语言风格**：
- 温和但有主见
- 偶尔俏皮

## 考研屋室友

### 吴彬

| 属性 | 描述 |
|-----|------|
| 目标 | 考研（法硕） |
| 标志 | 白衬衫永远笔挺、无袖线衫、黑外套、眼镜 |

**性格特点**：
- 考研屋最"精致"的一个
- 一本正经地说俏皮话
- 有"官腔"，像随时能去当书记员

**经典语录风格**：
```
"风是有形状了，脑子没了。"
"搞定。看见没，这就是气场。"
```

---

### 邓娜

| 属性 | 描述 |
|-----|------|
| 目标 | 考研（清华法硕） |
| 标志 | 荧光笔、丸子头、毛线帽、大嗓门 |

**性格特点**：
- 屋里最热闹的，八卦雷达灵敏
- 嘴硬心软，爱用夸张比喻
- 紧张时往嘴里塞面包，像仓鼠

**经典语录风格**：
```
"哎呀，这黄金不会姓徐吧？"
"呸！你这是典型的乌鸦嘴法硕派！"
```

### 张浩翔

| 属性 | 描述 |
|-----|------|
| 年龄 | 27岁（最年长） |
| 籍贯 | 广东 |
| 背景 | 在广州工作几年后辞职考研 |
| 目标 | 清华建筑学院（建筑艺术） |
| 标志 | 灰色冲锋衣裤、登山鞋、二胡 |

**性格特点**：
- 温温的，不急不慢
- 带着浪漫的倔强
- 是考研屋的"定海神针"

**语言风格**：
- 广普（广东普通话）
- 说话轻但有分量

**经典语录**：
```
"日子不可能一直像白粥那样，总得加点料。"
"真有意思，这题出的比青藏线还险。"
```
